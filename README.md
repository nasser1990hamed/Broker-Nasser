# Broker Nasser - وسيط ناصر

موقع ويب احترافي لـ **وسيط ناصر** - وسيط شامل في جميع المجالات.

## نظرة عامة

موقع ويب ثنائي اللغة (عربي/إنجليزي) يقدم خدمات الوساطة الشاملة في:
- الوساطة العقارية
- الوساطة التجارية
- الوساطة المالية
- وساطة السيارات
- وساطة المجوهرات والذهب
- الوساطة التجارية العامة

## التقنيات المستخدمة

- **React 19** + **TypeScript**
- **Tailwind CSS 4**
- **Vite**
- **shadcn/ui**
- **Framer Motion**

## أقسام الموقع

1. **الرئيسية (Hero)** - صورة خلفية بانورامية مع إحصائيات
2. **من نحن** - تعريف بوسيط ناصر وخبراته
3. **الخدمات** - عرض جميع خدمات الوساطة
4. **لماذا ناصر** - مميزات التعامل مع وسيط ناصر
5. **إنجازاتنا** - أرقام وإحصائيات الإنجازات
6. **آراء العملاء** - شهادات العملاء
7. **تواصل معنا** - نموذج التواصل ومعلومات الاتصال

## تشغيل المشروع

```bash
pnpm install
pnpm dev
```

## البناء للإنتاج

```bash
pnpm build
```
